## 残り作業

### 1. `.env` に追記
```
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### 2. PWA アイコン画像を用意
`public/icon-192.png` と `public/icon-512.png` を配置する。
- ゴルフ関連の PNG 画像（192×192px、512×512px）を用意して `public/` に置くだけ

### 3. Google Cloud Console にローカル用リダイレクト URI を追加
開発中のテスト用に以下を追加：
- 承認済みリダイレクト URI に `http://localhost:3000/api/auth/callback` を追加

### 4. 動作確認（STEP 11）
```
1. npm run dev で起動
2. http://localhost:3000 → /login にリダイレクトされることを確認
3. Googleログイン → 初回は /setup、2回目以降は /
4. /setup でHS・飛距離入力 → 番手テーブルが自動計算 → 保存
5. / → 「ピンをセット」→ マップタップ → localStorage に座標が入る
6. 「キャディに聞く」→ GPS・風速取得 → Claude提案がストリーミング表示
7. 「読み上げる」ボタンで音声再生
8. スマホのSafariで開き「ホーム画面に追加」→ アプリとして起動
```

### 5. Vercel デプロイ（任意）
```
1. vercel login
2. vercel --prod
3. 本番 URL を Google Cloud Console の承認済みリダイレクト URI に追加
4. Supabase → Authentication → URL Configuration に本番 URL を追加
5. Vercel の環境変数に .env の内容をすべて設定
```

## 注意事項

- `public/icon-192.png` と `public/icon-512.png` がないと PWA インストール時にアイコンが表示されない
- Supabase の `user_profiles`・`club_settings`・`subscriptions` テーブルは Prisma `db push` 済み
- RLS（Row Level Security）も設定済み
- Google OAuth は Supabase に設定済み（本番 URL 追加は Vercel デプロイ後に必要）
