# ゴルフキャディPWA - 実装手順

設計の詳細は [ゴルフキャディPWA_MVPプラン.md](./ゴルフキャディPWA_MVPプラン.md) を参照。

---

## 事前準備

### 1. Google Cloud Console でOAuthクライアント作成
1. [Google Cloud Console](https://console.cloud.google.com/) → APIとサービス → 認証情報
2. OAuthクライアントID作成（Webアプリケーション）
3. 承認済みリダイレクトURIに以下を追加：
   ```
   https://<supabase-project-id>.supabase.co/auth/v1/callback
   ```
4. クライアントID・シークレットをメモ

### 2. Supabase プロジェクト作成
1. [Supabase](https://supabase.com/) でプロジェクト作成
2. Authentication → Providers → Google を有効化
3. 手順1のクライアントID/シークレットを設定
4. Settings → API から `SUPABASE_URL` と `SUPABASE_ANON_KEY` をメモ

### 3. Anthropic APIキー取得
1. [Anthropic Console](https://console.anthropic.com/) でAPIキーを発行

### 4. `.env.local` を作成
```
NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...
ANTHROPIC_API_KEY=sk-ant-...
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

## STEP 7: ビジネスロジック

### `lib/clubs.ts`
- `DEFAULT_LOFTS`: 13番手分のロフト角プリセット
- `generateClubTable(headSpeed, driverDistance)`: 全番手の飛距離テーブルを生成
- `dbClubsToClubDistances()`: DB形式 → UI形式に変換

### `lib/wind.ts`
- `applyWindCorrection(distance, wind)`: 風補正計算

### `lib/distance.ts` ★
- `calcDistance(lat1, lng1, lat2, lng2)`: Haversine公式で2点間の距離をヤードで返す

```typescript
// Haversine公式の骨格
const R = 6371000 // 地球半径（メートル）
// ... 計算 ...
const meters = R * c
return meters * 1.09361 // メートル → ヤード
```

### `lib/speech.ts` ★
- `speak(text)`: SpeechSynthesisで日本語音声読み上げ
- ブラウザ非対応時はコンソールに警告を出してフォールバック

```typescript
export function speak(text: string) {
  if (typeof window === 'undefined' || !window.speechSynthesis) return
  const utterance = new SpeechSynthesisUtterance(text)
  utterance.lang = 'ja-JP'
  utterance.rate = 1.0
  window.speechSynthesis.cancel() // 前の読み上げを止める
  window.speechSynthesis.speak(utterance)
}
```

## STEP 8: API Routes

### `app/api/weather/route.ts`
- `GET ?lat=&lon=` で現在地の風速・風向を返す
- Open-Meteo API（無料・APIキー不要）を使用
- `next: { revalidate: 300 }` で5分キャッシュ

### `app/api/suggest/route.ts`
- `POST` でSSEストリーミング
- 受け取るパラメータ: `{ distance, windSpeed, windDirection, clubs }`
- 認証チェックに `getUser()` を使用（`getSession()` は使わない）
- モデル: `claude-opus-4-5`

## STEP 9: コンポーネント

### `components/ClubSetup.tsx`
- 番手テーブルを一覧表示
- 各番手の飛距離を手動調整できる入力フォーム

### `components/SuggestCard.tsx`
- SSEストリーミングの結果をリアルタイム表示
- カーソル点滅アニメーション付き

### `components/PinMap.tsx` ★
- `react-leaflet` でLeafletマップを表示
- タイルレイヤー: Esri World Imagery（衛星写真・無料）
  ```
  https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}
  ```
- マップタップ → ピンマーカーを設置 → 座標を親コンポーネントに渡す
- SSR対策: `dynamic(() => import(...), { ssr: false })` でインポート

## STEP 10: ページ

### `app/setup/page.tsx`
1. ヘッドスピード・ドライバー飛距離を入力
2. `generateClubTable()` でリアルタイム計算してプレビュー
3. `ClubSetup` で手動調整
4. 保存: `user_profiles` に upsert → `club_settings` に insert
5. 完了後 `/` へ遷移

### `app/pin/page.tsx` ★
1. `PinMap` を表示（初期位置は現在地のGPS座標）
2. タップしたピン座標を state で管理
3. 「セット完了」ボタン押下 → `localStorage.setItem('pinLat', ...)` で保存
4. `/` へ遷移

### `app/page.tsx` ★
UIは最小限（ボタン2つ）:

```
[ ピンをセット ]      ← /pin へ遷移

[ キャディに聞く ]    ← メインアクション
```

「キャディに聞く」の処理順序：
1. `navigator.geolocation.getCurrentPosition()` で現在地取得
2. `/api/weather?lat=&lon=` で風速・風向取得
3. `localStorage` からピン座標を取得
4. `calcDistance()` で距離をヤード計算
5. `/api/suggest` にPOST
6. SSEをストリーミングで受信 → `SuggestCard` に表示
7. ストリーミング完了後に `speak()` で音声読み上げ

## STEP 11: 動作確認

1. `npm run dev` で起動
2. `http://localhost:3000` → `/login` にリダイレクトされる
3. Googleログイン → 初回は `/setup`、2回目以降は `/`
4. `/setup` でHS・飛距離入力 → 番手テーブルが自動計算される
5. 保存 → Supabaseに `user_profiles` と `club_settings` が作成される
6. `/` → 「ピンをセット」→ マップタップ → localStorage に座標が入る
7. 「キャディに聞く」→ 提案がストリーミング表示され音声で読み上げられる
8. スマホのSafariで開き「ホーム画面に追加」→ アプリとして起動できる
